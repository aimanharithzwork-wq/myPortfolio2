Final Year Project (FYP) Paper

Title: Fitness Tracker Mobile Application for Calisthenics Exercises

Published By/In: Fitness Tracker Mobile Application for Calisthenics Exercises – Published in Journal of Computing Research and Innovation (JCRINN), Vol. 8 (2), 2023.

This project presents a mobile application titled *“Fitness Tracker Mobile Application for Calisthenics Exercises”, aimed at helping users track and manage their bodyweight workout routines. The main objective is to provide a simple and efficient platform for monitoring calisthenics exercises and encouraging consistent physical activity.

The system was developed using the Software Development Life Cycle (SDLC), involving requirement analysis, design, implementation, and testing. Key features include workout tracking, exercise scheduling, and progress monitoring, supported by a database for secure data storage.

Findings show that the application improves user motivation and helps users maintain regular exercise habits. Overall, the system proves effective as a practical fitness tracking tool for calisthenics training.
